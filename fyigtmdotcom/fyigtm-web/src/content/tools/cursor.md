Cursor is an AI-first code editor built for pair programming with AI, designed to make developers significantly more productive.

## What is Cursor?

Cursor is a code editor forked from VS Code that puts AI assistance at its core. It features deep AI integration for code generation, editing, and understanding, going beyond simple autocomplete.

## Key Features

- AI-powered code generation
- Natural language code editing
- Codebase-aware suggestions
- Built-in chat interface
- VS Code extension compatibility
- Multi-file editing support

## Pricing

Cursor offers a free tier with limited AI usage. Pro at $20/month provides unlimited AI features and priority access.
