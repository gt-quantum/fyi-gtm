Perplexity is an AI-powered answer engine that provides direct, sourced answers to your questions instead of just links.

## What is Perplexity?

Perplexity reimagines search by providing conversational, cited answers to your questions. It combines large language models with real-time web search to deliver accurate, up-to-date information.

## Key Features

- Direct answers with citations
- Real-time web search integration
- Follow-up question support
- Multiple AI model options
- Focus modes for different use cases
- Collections for organizing research

## Pricing

Perplexity offers a generous free tier. Perplexity Pro at $20/month provides access to more powerful models and unlimited Pro searches.
