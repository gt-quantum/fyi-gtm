Claude is an AI assistant created by Anthropic, designed to be helpful, harmless, and honest. It excels at analysis, writing, coding, and complex reasoning tasks.

## What is Claude?

Claude is a next-generation AI assistant built for work and trained to be safe, accurate, and secure. It can handle complex, multi-step instructions and help with a wide variety of tasks from summarization to coding.

## Key Features

- Advanced reasoning and analysis capabilities
- Long context window for processing large documents
- Strong coding and technical assistance
- Thoughtful and nuanced responses
- Built-in safety measures
- Document and image analysis

## Pricing

Claude offers a free tier with usage limits. Claude Pro is available at $20/month with higher limits and priority access.
